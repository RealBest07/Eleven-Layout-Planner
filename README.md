# 🏪 7-Eleven Layout Planner

เครื่องมือวางผังร้าน 7-Eleven และคำนวณที่จอดรถ บนเว็บ (HTML + Canvas 2D, ไฟล์เดียว ไม่มี build/backend)

## ฟีเจอร์
- โหมดที่ดิน **สี่เหลี่ยม** (auto-layout) และ **หลายเหลี่ยม** (วาดเอง, หลายแปลง, ลากอาคาร)
- คำนวณช่องจอด + ระยะถอย + หมอนจอด + ทางเดิน อัตโนมัติ
- **เทมเพลตร้าน**: แบบมาตรฐาน + วาดเอง (polygon) + อัปโหลดรูปแปลน — เก็บใน localStorage
- Export PNG / PDF

## ใช้งาน
เปิด `index.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร
